Detailed Description

ISO-639 language code.

A 2 or 3 lower case language code selected from the following iso standards in this order:

  1. ISO 639-1 (2002)
  2. ISO 639-2 (1998)
  3. ISO 639-3 (2007)
  4. ISO 639-5 (2008),
