Detailed Description

A Uniform Resource Identifier.

This class holds the URI separated and unencoded into it components:

  • scheme (optional)
  • authority (optional)
    • userinfo (optional)
    • host
    • port (optional)
  • path
  • query (optional)
  • fragment (optional)
