Detailed Description

A set of glyph-ids of a font which composites into a single glyph.

The normal operation for getting a glyph-ids is by:

  • By looking up a non-typographical-ligature grapheme in a font, returning 1 or more glyphs representing that single grapheme.
  • By morphing a sequence of glyph_ids objects into new glyph_ids objects, where some of the glyph_ids objects may get merged into a ligature of multiple graphemes.
