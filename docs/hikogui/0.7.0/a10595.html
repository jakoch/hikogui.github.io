Internationalization

Translatable strings

The hi::tr() constructor creates an object containing the translatable format string and its arguments. The gettext tool can be used to extract these strings from the application.

The first integer argument of hi::tr the constructor is used to select the plural translation.

Widgets will call the hi::tr object to translate and format the string for the current configured language and locale.
